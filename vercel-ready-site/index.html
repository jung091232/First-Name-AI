프로젝트 소개

First Name AI는 Wallet · Game · NFT · Token · Mining · DAO를 하나로 연결한 Web3 통합 생태계입니다.

생태계 전체 흐름

하나의 지갑에서 시작해, NCT 토큰을 활용한 로또, 쇼핑, 채굴, NFT 배당, DAO 투표까지 모든 활동이 순환 구조로 연결됩니다.

  • Wallet → NCT 토큰 → Crypto Lotto Game
  • Crypto Lotto → NCT 회수·분배 → NFT 배당 재원
  • NCT → 쇼핑몰 결제, 채굴기 구매, 수수료(GAS) 사용
  • 수익 일부 → NFT 보유자 배당 · DAO 운영 재원

참여 방법

Name Wallet 설치부터 패키지 선택, 로또 참여, NFT 배당까지 단계별로 간단하게 참여할 수 있습니다.

STEP 1
Name Wallet 설치
공식 웹사이트(namewallet.io) 또는 안내된 링크를 통해 Name Wallet 앱을 다운로드하고 설치합니다.
STEP 2
회원가입 & 추천코드 입력
지갑 생성 후, 회원가입을 진행하고 추천인이 있을 경우 추천코드를 입력합니다.
STEP 3
NFT/패키지 선택
원하는 금액 구간($100~$3,000 / $3,000~$10,000 / $10,000 / $20,000 등)을 선택해 NFT 패키지를 구매합니다.
STEP 4
결제 및 NCT 지급 확인
지정된 결제 수단으로 결제를 완료하면, 해당 금액에 따라 NCT 코인 및 NFT가 발급·지급됩니다.
STEP 5
Crypto Lotto · 쇼핑 · 서비스 이용
지갑 내에서 크립토 로또, 쇼핑몰, 게임, 채굴기 등 다양한 서비스를 이용합니다.
STEP 6
보상 및 배당 수령
활동에 따라 적립되는 보상과 NFT 배당 수익을 정기적으로 확인하고, 지갑에서 바로 관리할 수 있습니다.

NFT 배당 구조

배당형 NFT는 게임 수익, 지갑 수수료, 쇼핑몰 수익 등에서 발생한 일부 수익을 NFT 보유자에게 재분배하는 구조로 설계됩니다.

  • Crypto Lotto, 지갑 수수료, 쇼핑몰, 채굴기 판매 등에서 발생하는 수익 일부 배분
  • 일반 / VIP / VVIP 등 NFT 등급에 따라 배당 비율 및 기간 차등
  • 기본 구조: 2년(일반) / 3년(VIP, VVIP) 배당
