회사 소개

First Name AI는 블록체인 기반 Web3 플랫폼 기업으로, 전자지갑·크립토 로또·NFT·채굴기·DAO 구조를 하나의 생태계로 통합하는 프로젝트입니다.

2023년 설립 이후, 베트남 호치민을 중심으로 지갑과 로또 게임, NCT 코인, 배당형 NFT, DIGGER 채굴 디바이스 등 다양한 모듈을 개발해 왔습니다.

  • 2023년 2H – 베트남 본사 설립, 블록체인 전문 개발사 파트너십 체결
  • 2024년 1H – 로또 게임 및 지갑 개발 초기 설계, 라이센스 및 개발사 선정
  • 2024년 2H – 지갑 · 로또 게임 베타버전 테스트, 기능 추가 개발 진행
  • 2025년~ – 전체 개발 과정 일괄화, NCT 코인 상장 및 생태계 확장

프로젝트 소개

First Name AI는 Wallet · Crypto Lotto · NCT 토큰 · NFT 배당 · DIGGER 채굴기 · DAO를 하나로 연결한 Web3 통합 생태계입니다.

ECOSYSTEM FLOW

하나의 지갑에서 시작해, NCT 토큰을 활용한 로또, 쇼핑, 채굴, NFT 배당, DAO 투표까지 모든 활동이 순환 구조로 연결됩니다.

  • Wallet → NCT 토큰 → Crypto Lotto Game
  • Crypto Lotto → NCT 회수·분배 → NFT 배당 재원
  • NCT → 쇼핑몰 결제, 채굴기 구매, 수수료(GAS) 사용
  • 수익 일부 → NFT 보유자 배당 · DAO 운영 재원
DAO CORE

DAO 및 시스템 구조

DAO 기반 거버넌스를 통해 참여자들이 프로젝트 방향성에 직접 영향을 미치고, 지갑·게임·채굴·배당 구조를 투명하게 관리하는 것을 목표로 합니다.

GAME

Crypto Lotto Game

매회차 5,000장 한정 티켓으로 운영되는 크립토 로또 게임입니다. 하루 20~30회 추첨 구조를 통해 반복성과 고빈도 게임성을 제공합니다.

TOKEN

NCT 유틸리티 코인

NCT는 로또 티켓, 쇼핑몰 결제, 지갑 수수료, 채굴기 구매, NFT 배당 재원 등 다양한 영역에서 사용되는 실사용 유틸리티 토큰입니다.

MINING

DIGGER 채굴 디바이스

가정용으로도 설치 가능한 AI 기반 채굴 디바이스로, POS 구조와 연계해 안정적인 채굴 참여를 돕습니다.

보상 플랜

First Name AI는 추천, 바이너리, 글로벌 풀, 랭크, NFT 배당 등 다양한 보상 구조를 통해 생태계 참여 인센티브를 제공합니다.

DIRECT BONUS

참여 방법

Name Wallet 설치부터 패키지 선택, Crypto Lotto 참여, NFT 배당까지 단계별로 간단하게 참여할 수 있습니다.

STEP 1

Name Wallet 설치

공식 안내 링크를 통해 Name Wallet을 다운로드 후 설치합니다.

STEP 2

회원가입 및 추천코드 입력

지갑 생성 및 회원가입을 진행하고, 추천인이 있을 경우 코드를 입력합니다.

STEP 3

패키지 및 NFT 선택

원하는 금액 구간의 NFT 패키지를 선택해 참여합니다.

STEP 4

결제 및 NCT 지급

결제 후 해당 금액에 따라 NCT 및 NFT가 지갑으로 지급됩니다.

STEP 5

Crypto Lotto · 쇼핑 · 채굴 참여

로또·쇼핑몰·채굴기 등 다양한 Web3 서비스를 이용합니다.

STEP 6

보상 및 배당 수령

활동에 따라 적립되는 보상과 NFT 배당 수익을 지갑에서 관리합니다.

NFT 배당 구조

배당형 NFT는 로또 수익, 지갑 수수료, 쇼핑몰 수익 등에서 발생한 일부 수익을 NFT 보유자에게 분배하는 구조로 설계됩니다.

DIVIDEND FLOW

일반 · VIP · VVIP NFT는 회사 전체 수익의 일부를 2년 또는 3년 동안 배당받는 구조로 설계되어 있습니다.

NFT 등급별 예시

구분 NFT 금액 월 예상 수익 기간 예상 총 수익
일반 NFT $3,000 $800 ~ $1,200 2년 $19,200 ~ $28,800
VIP NFT $10,000 $1,800 ~ $3,000 3년 $64,800 ~ $108,000
VVIP NFT $20,000 $3,000 ~ $5,000 3년 $108,000 ~ $180,000
